# Smart Robot Arduino Code

Bu klasör, Smart Obstacle Avoidance Robot projesinin Arduino Nano mikrodenetleyicisi üzerinde çalışan kodlarını içerir.

## Özellikler
- HC-05 Bluetooth Modülü üzerinden haberleşme.
- Otonom (Engelden Kaçan) ve Manuel sürüş modları.
- Sensörler ve motorlar arası tam senkronizasyon.

## Klasör Yapısı
- `SmartRobot.ino`: Robotun ana C++ kodu. Bu kod Arduino IDE ile derlenip yüklenebilir.
